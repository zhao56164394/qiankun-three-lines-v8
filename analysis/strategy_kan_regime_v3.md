### 卖点逻辑

```python
def sell_kan_regime_v3(buy_idx, td_seg, max_end):
    """坎 regime v3 卖点: bull 单独"""
    cross_count = 0
    running_max = td_seg[buy_idx]

    for k in range(buy_idx + 1, max_end + 1):
        if not isnan(td_seg[k]):
            running_max = max(running_max, td_seg[k])

        if k > 0 and running_max >= 89 \
           and td_seg[k] < 89 and td_seg[k-1] >= 89:
            cross_count += 1
            if cross_count == 2:
                return k, 'bull_2nd'

    return max_end, 'timeout'
```

## 6. 完整业务故事

> **坎 regime** (Gate 1): 大盘长期熊未尽, 中期已经反弹了一波, 短期在洗盘 — 这是反弹后的二次启动窗口, 浮筹换手干净, 主力悄悄介入。
>
> **个股巽日** (Gate 2): 长期低位 + 中短期已反弹, 反转的火苗刚冒头。74.2% 的主升浪起点前一日就是巽日, 这是反转入口的强普世规律。
>
> **避雷** (Gate 3): 排除"伪启动"形态:
>   - 大盘月线在初动 (震) 或中期顶 (兑) — 跟坎 regime 不协同
>   - 个股年卦在乾 / 月卦在兑 — 已经涨完
>
> **排名** (Gate 4): 真正的好买点要么是:
>   1. 大盘月线也在反转 (巽) — 共振信号 (综合 lift +40)
>   2. 大盘日卦止涨 (艮) — 反直觉但跨段稳
>   3. 主力极弱后回升 (mf_5d<-50) — 浮筹清洗到位
>   4. 当日主力发力 (mf>100) — 入场确认
>   5. 散户极度低迷 (sanhu_5d<-100) — 反向情绪
>
> **持仓**: 等到 d_trend 第 2 次下穿 89 (主升浪走完) 卖出。坎 regime 假突破跌不深, 不需要时间止损。

## 附录 B: 实施伪代码 (买卖完整版)

```python
def kan_regime_buy_signal(stock, day):
    """v3 买点判断"""
    # Gate 1: regime
    if market_y_gua(day) != '010': return None  # 坎

    # Gate 2: 时机
    if stock.d_gua(day) != '011': return None  # 巽

    # Gate 3: 硬避雷
    if market_m_gua(day) in ['100', '110']: return None  # 大盘月线震/兑
    if stock.y_gua(day) == '111': return None  # 个股年卦乾
    if stock.m_gua(day) == '110': return None  # 个股月卦兑

    # Gate 4: 软排名 score
    score = 0
    if market_m_gua(day) == '011': score += 1   # 大盘月线巽 (协同共振)
    if market_d_gua(day) == '001': score += 1   # 大盘日卦艮
    
    mf = stock.main_force(day)
    mf_5d = stock.main_force_5d_mean(day)
    sanhu_5d = stock.retail_5d_mean(day)
    
    if mf_5d < -50: score += 1   # 主力极弱回升前夜
    if mf > 100: score += 1      # 主力发力
    if sanhu_5d < -100: score += 1  # 散户极弱

    if score < 2: return None

    return {
        'code': stock.code,
        'date': day,
        'price': stock.close(day),
        'score': score,
    }


def kan_regime_sell_signal(stock, buy_idx, hold_max=60):
    """v3 卖点判断 (bull 单独)"""
    cross_count = 0
    running_max = stock.d_trend(buy_idx)
    end_idx = buy_idx + hold_max

    for k in range(buy_idx + 1, end_idx + 1):
        td_k = stock.d_trend(k)
        if not isnan(td_k):
            running_max = max(running_max, td_k)

        td_prev = stock.d_trend(k - 1)
        if running_max >= 89 and td_k < 89 and td_prev >= 89:
            cross_count += 1
            if cross_count == 2:
                return k, 'bull_2nd', stock.close(k)

    return end_idx, 'timeout', stock.close(end_idx)
```
