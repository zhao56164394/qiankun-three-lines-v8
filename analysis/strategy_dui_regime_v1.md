# 兑 regime 主升浪策略 v1

**版本**: v1.0
**日期**: 2026-04-28
**适用**: 大盘 y_gua = 110 兑 期间, 个股 d_gua = 000 坤 当日
**状态**: ⚠️ **样本薄可投产** (仅 2 段 w2/w5, 但 2/2 段都正 alpha, 含大熊市段 +3.64%)

⚠️ **主要限制**: 兑 regime 历史只在 w2_2019 (大牛顶) 和 w5_2022 (大熊初) 出现, 跨 8 年只 2 段.
没有 IS/OOS 严格拆分能力, 任何额外细化都是切片福利风险.

## 2. 大盘环境与样本

### 大盘卦象定义

```
兑 = y_pos=1 (长期高) + y_spd=1 (中期高) + y_acc=0 (短期低)

业务含义: "长期上涨 + 中期延续 + 短期已衰退"
        — 顶部初动 / 高位震荡顶
        — 跟乾 (普涨抱团) 不一样: 短期已经开始落
```

### 历史窗口

```
样本期 2018-2026, 兑 regime 仅出现 2 段:
  w2_2019:  285,352 全市场事件 (任何卦) | baseline 30日 +2.79%, 大牛市顶
  w5_2022:   45,505 全市场事件             | baseline 30日 -11.11%, 大熊市初

总: 330,857 (任何触发卦, 在兑 regime 内)
```

⚠️ 其他 5 段 (w1/w3/w4/w6/w7) 完全没有兑 regime 的数据.

### 跟乾 regime 的对比 — 关键反差

| 维度 | 乾 regime (普涨期) | 兑 regime (顶部衰退) |
|---|---|---|
| 大盘 y_gua | 111 | 110 |
| 业务 | 全面抱团/普涨 | 顶部初动/衰退 |

兑 regime 是顶部衰退期, 巽日反弹容易"反转无效", 反而坤日 (个股短期已落) 是已经吃过回调、修复概率高.

### 卖点逻辑

```python
def sell_dui_regime_v1(buy_idx, td_seg, max_end):
    """兑 regime v1 卖点: bull 单独"""
    cross_count = 0
    running_max = td_seg[buy_idx]

    for k in range(buy_idx + 1, max_end + 1):
        if not isnan(td_seg[k]):
            running_max = max(running_max, td_seg[k])

        if k > 0 and running_max >= 89 \
           and td_seg[k] < 89 and td_seg[k-1] >= 89:
            cross_count += 1
            if cross_count == 2:
                return k, 'bull_2nd'

    return max_end, 'timeout'
```

## 11. 实施伪代码 (买卖完整版)

```python
def dui_regime_buy_signal(stock, day, market):
    """v1 买点判断 (4 道关卡)"""
    # Gate 1: regime
    if market.y_gua(day) != '110': return None  # 兑

    # Gate 2: 时机 (反直觉: 用坤, 不是巽)
    if stock.d_gua(day) != '000': return None  # 坤

    # Gate 3: 卦象避雷
    if market.d_gua(day) == '011': return None  # 大盘日巽
    if stock.m_gua(day) in {'001', '011', '101', '111'}: return None  # 个股月: 艮/巽/离/乾

    # 没有软排名 score (兑 regime 没有 2/2 双 pass 信号)

    return {
        'code': stock.code,
        'date': day,
        'price': stock.close(day),
    }


def dui_regime_sell_signal(stock, buy_idx, hold_max=60):
    """v1 卖点判断 (bull 单独)"""
    cross_count = 0
    running_max = stock.d_trend(buy_idx)
    end_idx = buy_idx + hold_max

    for k in range(buy_idx + 1, end_idx + 1):
        td_k = stock.d_trend(k)
        if not isnan(td_k):
            running_max = max(running_max, td_k)

        td_prev = stock.d_trend(k - 1)
        if running_max >= 89 and td_k < 89 and td_prev >= 89:
            cross_count += 1
            if cross_count == 2:
                return k, 'bull_2nd', stock.close(k)

    return end_idx, 'timeout', stock.close(end_idx)
```
