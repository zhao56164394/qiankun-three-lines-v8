## 5. 卖点 — V9 bull + TS20 (Step 34 验证最优)

### 退出规则 (3 道防线)

```python
def sell_kun_regime_v3(buy_idx, td_seg, max_end):
    """坤 regime 巽日策略卖点 v3"""
    cross_count = 0
    running_max = td_seg[buy_idx]

    for k in range(buy_idx + 1, max_end + 1):
        if not isnan(td_seg[k]):
            running_max = max(running_max, td_seg[k])

        # 主路: 第 2 次下穿 89 → 卖
        if k > 0 and running_max >= 89 \
           and td_seg[k] < 89 and td_seg[k-1] >= 89:
            cross_count += 1
            if cross_count == 2:
                return k, 'bull_2nd'

        # 保险: 持仓 ≥ 20 日 且 d_trend 从未到过 89 → 强卖
        if k - buy_idx >= 20:
            if max(td_seg[buy_idx:k+1]) < 89:
                return k, 'time_stop_20d'

    # 兜底: 60 日强卖
    return max_end, 'timeout'
```

### 退出类型分布 (V9 bull+TS20 实证)

| 类型 | 占比 | 平均收益 | 业务含义 |
|---|---|---|---|
| **bull_2nd** (第 2 次穿 89) | 20% | 走完主升浪 | ★ 让赚钱的多跑 |
| **time_stop_20d** (20日时间止损) | 49% | 假突破止损 | 止损 -6.4%, 不熬亏 |
| **timeout** (60 日兜底) | 32% | 中性 | 既没起来也没死透 |

### V9 bull+TS20 的设计哲学

> **bull 路线**: 让真主升浪充分跑完 (等第 2 次穿 89, 不在第 1 次出)
> **TS20 保险**: 但给假突破一个观察期 (20 日)
>   - 20 日内 d_trend 从未到过 89 → 这股就是没起来, 强卖止损
>   - 20 日内到过 89 (主升浪启动了) → 继续等第 2 次穿 89

## 6. 完整业务故事

> **坤年环境** (Gate 1): 大盘三尺度全低, 长期熊市末段或震荡底, 多数股被错杀, 是好做多的环境。
>
> **个股巽日** (Gate 2): 长期低位 + 中短期已反弹, 反转的火苗刚冒头 — 不是底部一炮起, 是底部反弹的第一波中继。
>
> **避雷** (Gate 3): 排除 9 种"伪反弹"形态 — 个股月线已走完 (离/兑/乾) / 个股年卦没空间 (艮/巽) / 大盘日卦还没启动 (坤/艮/震/离)。
>
> **排名** (Gate 4): 真正的好买点要么是**大盘月线刚启动反转 (震)**, 要么是**大盘日卦协同 (巽)**, 要么是**主力当日已入场 (mf>100)**, 要么是**个股月线洗盘干净 (坎)**。命中越多, 越值得买。
>
> **持仓** (§5 卖点): 入场后给它机会跑 (等第 2 次下穿 89), 但若 20 日还没启动 (d_trend 从未到 89), 就快速止损不熬。

## 附录 A: 数学定义

### 个股 d_gua = 011 巽 计算
```
d_pos = (d_trend ≥ 50)
d_spd = (d_trend - prev_d_trend) ≥ SPD_HYST  (默认 2.0)
d_acc = ABS(d_mf - prev_d_mf) ≥ ACC_HYST     (默认 30.0)

011 巽 = 天位低 + 人位高 + 地位高
       = d_pos=0 (long-trend low) + d_spd=1 + d_acc=1
       (具体三爻定义见 data_layer/prepare_stock_multi_scale_gua.py)
```

### 大盘 y_gua / m_gua / d_gua
```
跟个股同公式, 但用大盘指数计算 (000001.SH)
```

### main_force (主力线)
```
日级资金流量动量指标 (具体公式见 stocks 数据层)
正值 = 主力净流入, 负值 = 净流出
```

### d_trend (趋势线)
```
日级趋势动量指标, 范围 [0, 100]
≥89 = 极强势 (顶部区间)
≤11 = 极弱势 (底部区间)
```

## 附录 B: 实施伪代码 (买卖完整版)

```python
def kun_regime_buy_signal(stock, day):
    """v3 买点判断"""
    # Gate 1: regime
    if market_y_gua(day) != '000':  # 坤
        return None

    # Gate 2: 时机
    if stock.d_gua(day) != '011':  # 巽
        return None

    # Gate 3: 硬避雷
    if stock.m_gua(day) in ['101', '110', '111']: return None
    if stock.y_gua(day) in ['001', '011']: return None
    if market_d_gua(day) in ['000', '001', '100', '101']: return None

    # Gate 4: 软排名 score
    score = 0
    if market_m_gua(day) == '100': score += 1   # 震
    if market_d_gua(day) == '011': score += 1   # 巽
    if stock.main_force(day) > 100: score += 1  # 主力发力
    if stock.m_gua(day) == '010': score += 1   # 坎

    if score < 2:
        return None

    return {
        'code': stock.code,
        'date': day,
        'price': stock.close(day),
        'score': score,
    }


def kun_regime_sell_signal(stock, buy_idx, hold_max=60):
    """v3 卖点判断 (V9 bull + TS20)"""
    cross_count = 0
    running_max = stock.d_trend(buy_idx)
    end_idx = buy_idx + hold_max

    for k in range(buy_idx + 1, end_idx + 1):
        td_k = stock.d_trend(k)
        if not isnan(td_k):
            running_max = max(running_max, td_k)

        # 主路: 第 2 次下穿 89
        td_prev = stock.d_trend(k - 1)
        if running_max >= 89 and td_k < 89 and td_prev >= 89:
            cross_count += 1
            if cross_count == 2:
                return k, 'bull_2nd', stock.close(k)

        # 保险: 20 日 d_trend 仍未到 89
        if k - buy_idx >= 20:
            seg_max = max(stock.d_trend(j) for j in range(buy_idx, k + 1))
            if seg_max < 89:
                return k, 'time_stop_20d', stock.close(k)

    # 兜底
    return end_idx, 'timeout', stock.close(end_idx)
```
