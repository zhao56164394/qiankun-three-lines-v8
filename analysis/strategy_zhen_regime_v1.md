震 regime = 长期高 + 中期低 + 短期低 = 高位深度回调期.
- **巽日** (个股反转) → 在中短都低的环境里"反转无效", 容易追到假反弹
- **坤日** (个股短期低) → 还在跟着大盘跌, 没修复
- **坎日** (个股短期高 + 中期低 + 长期低) → 个股已开始反弹但底位, 跟震 regime 形成"补涨" 关系 ⭐

### 卖点逻辑

```python
def sell_zhen_regime_v1(buy_idx, td_seg, max_end):
    """震 regime v1 卖点: bull 单独"""
    cross_count = 0
    running_max = td_seg[buy_idx]

    for k in range(buy_idx + 1, max_end + 1):
        if not isnan(td_seg[k]):
            running_max = max(running_max, td_seg[k])

        if k > 0 and running_max >= 89 \
           and td_seg[k] < 89 and td_seg[k-1] >= 89:
            cross_count += 1
            if cross_count == 2:
                return k, 'bull_2nd'

    return max_end, 'timeout'
```

## 12. 实施伪代码 (买卖完整版)

```python
def zhen_regime_buy_signal(stock, day, market):
    """v1 买点判断 (4 道关卡)"""
    # Gate 1: regime
    if market.y_gua(day) != '100': return None  # 震

    # Gate 2: 时机 (反直觉: 用坎, 不是巽/坤)
    if stock.d_gua(day) != '010': return None  # 坎

    # Gate 3: 弱避雷
    if market.d_gua(day) in {'101', '111'}: return None  # 大日离/乾
    if stock.y_gua(day) == '111': return None  # 股年乾

    # Gate 4: 软排名 score ≥ 1
    score = 0
    if market.d_gua(day) == '011': score += 1  # 大日巽
    if stock.m_gua(day) == '110': score += 1   # 股月兑
    if score < 1: return None

    return {
        'code': stock.code,
        'date': day,
        'price': stock.close(day),
        'score': score,
    }


def zhen_regime_sell_signal(stock, buy_idx, hold_max=60):
    """v1 卖点判断 (bull 单独)"""
    cross_count = 0
    running_max = stock.d_trend(buy_idx)
    end_idx = buy_idx + hold_max

    for k in range(buy_idx + 1, end_idx + 1):
        td_k = stock.d_trend(k)
        if not isnan(td_k):
            running_max = max(running_max, td_k)

        td_prev = stock.d_trend(k - 1)
        if running_max >= 89 and td_k < 89 and td_prev >= 89:
            cross_count += 1
            if cross_count == 2:
                return k, 'bull_2nd', stock.close(k)

    return end_idx, 'timeout', stock.close(end_idx)
```
